Cryptsy Trader

This is an android application that me and my partner Seth made as a class project. Cryptsy Trader. This is a crypto currency app that interfaces with the Cryptsy API. The app allows you to check your balances, order history, active orders, and allows you to place orders. This was done by interfacing with the Cryptsy website through HTTP post. It uses your key and secrete to authenticate, and these can be set using the QR scanner.

TRP

I have made a small sample game called TRP. This is a top down dungeon dive game composed in the console using ASCII graphics. The Hero can be named and has three stats: Health, Sword Power, and Magic Power. The game allows the user to maneuver the character through a 2-dimensional character array and displays the content to the screen.

GM Tools

I have an app in progress for the Android Gm Tools. The tools will keep track of the characters initiative. It will store basic information about the characters. It will have a text random dice roller.
