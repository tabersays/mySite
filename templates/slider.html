This is a work in progress of a slider widget. This widget is using AngularJS and is not implementing JQuery. That said it is still under progress. This widget will be part of a collection of widgets from the Bossy UI project which can be found here. Go check it out!



It is very easy to use this slider. Once you get set up with AngularJS and incorporate our slider directive you can use it by simply using a tag like normal html. ex.

<bossy-slider baremptycolor="#550055" min="0" max="100" step="15" ng-model="valueToUse"></bossy-slider>

It can alternatively be written as an attribute to a tag. ex.

<div bossy-slider baremptycolor="#550055" min="0" max="100" step="15" ng-model="valueToUse"></div>

This creates the slider. We are using the ng-model to bind the value of the slider to a variable called valueToUse. This we can use interpolation to get the value. ex. double curly braces.
